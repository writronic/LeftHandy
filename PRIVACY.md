# Privacy Policy

**Last Updated:** June 2026

## Overview

LeftHandy is a free, open-source macOS menu bar application for left-handed mouse and cursor customization. **LeftHandy does not collect, store, or transmit any personal data.**

The full source code is publicly available at [github.com/writronic/LeftHandy](https://github.com/writronic/LeftHandy).

---

## Data Collection

LeftHandy does **not** collect:

- Personal information (name, email, etc.)
- Usage analytics or telemetry
- Crash reports or diagnostics
- Device identifiers or location data

---

## Local Data

All data stays on your device:

| Data | Location | Purpose |
|------|----------|---------|
| Preferences | `com.writronic.lefthandy` domain | Remember your settings |
| Login item | Managed by macOS via `SMAppService` | Optionally activate at startup |

---

## Network Activity

LeftHandy makes **zero network requests**. It operates entirely offline with no external connections.

---

## Third-Party Dependencies

No analytics, advertising, tracking, or update SDKs are included.

---

## Data Deletion

All data is stored locally and can be removed by:

- Resetting preferences via `defaults delete com.writronic.lefthandy`
- Uninstalling LeftHandy
